# Student Management System

This is a simple console-based Student Management System developed using Python.

## Features
- Add Student
- View Students
- Delete Student
- Exit Program

## Technologies Used
- Python

## How to Run
1. Download the project files
2. Open the file in Python or VS Code
3. Run the program

## Project Description
This project allows users to manage student records using a console application. 
Users can add new students, view the list of students, and delete student records.
